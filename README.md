# fizzbuzz
Implements the rules of Fizz Buzz game: a user gives a number (or many) and expects a reply with numbers divisible by three replaced with the word "fizz" and numbers divisible by five with the word "buzz".
```json
{
  "numbers": [
    1,
    2,
    3,
    15,
    45
  ]
}
```
```json
{
  "results": [
    "1",
    "2",
    "fizz",
    "fizzbuzz",
    "fizzbuzz"
  ]
}
```
